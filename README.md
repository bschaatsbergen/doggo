# doggo

Get your ASCII dog fix.

## Install

```sh
go install github.com/bschaatsbergen/doggo@latest
```

## Usage

```sh
$ doggo -h
doggo - go get a dog

Usage:
  doggo

Flags:
  -c, --copy   copy ASCII dog to clipboard
  -h, --help   help for doggo
```

## Get a dog

```sh
$ doggo --copy

    / \__
   (    @\___
   /         O
 /   (_____/
/_____/   U
```
