# doggo

```

   (    @\___
   /         O
 /   (_____/
/_____/   U
```

doggo is a simple command line tool to get a dog. It prints an ASCII dog to the terminal. Optionally, it can copy the dog to the clipboard.

## Installing

From source:
```sh
git clone git@github.com:bschaatsbergen/doggo
cd doggo
make
```

## Usage

```sh
doggo - go get a dog

Usage:
  doggo

Flags:
  -c, --copy   copy ASCII dog to clipboard
  -h, --help   help for doggo
```
